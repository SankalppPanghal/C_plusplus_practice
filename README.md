# C++ practice

### Random C++ Practice codes

This repository is a collection of miscellaneous .cpp files — small experiments, random ideas, and code snippets.

### Disclaimer

Nothing here is meant to be taken too seriously.

Code quality may vary wildly.
Some files exist purely out of curiosity.
Others were written just to see what happens.

### About the Code

Major parts of this repository were created with assistance from Gemini, to help me practice with C++.

### Structure

There is no strict structure. <br><br>
There are some leetcode style questions for files starting with **"q1_cpp"** to **"q12_"** whereas some files start with **"practice_"** touching the particular topic.

Each .cpp file is generally:

Self-contained (most of the time).<br>
Focused on a specific idea or test.<br>
Not guaranteed to be reusable or production-ready.<br>

### How to Run

If you really want to run something:

clang++ file.cpp -o output <br>
./output
